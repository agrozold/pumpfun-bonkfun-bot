# Сессия 004: Оптимизация Whale Tracker

**Дата:** 2026-01-19
**Статус:** Завершено

## Проблема

Whale tracker переподключался каждую минуту из-за таймаутов WSS.
Это вызывало избыточные запросы к Chainstack (302k/час).

## Решение

Увеличены таймауты в whale_tracker.py:
- WSS receive timeout: 120s → 180s
- Message processing timeout: 60s → 90s

## Результаты

| Метрика | До | После |
|---------|-----|-------|
| Запросов/час | 302,053 | 77,371 |
| Снижение | - | **75%** |
| Reconnects | каждую минуту | стабильно |

## Файлы изменены

- src/monitoring/whale_tracker.py (таймауты)
